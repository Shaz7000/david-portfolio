# David Portfolio

Next.js 14 (App Router) + Tailwind CSS + Framer Motion.

## Prerequisites

**Node.js 18.18+ required.** Your current Node version is too old. Install the latest LTS from https://nodejs.org/ (or via `winget install OpenJS.NodeJS.LTS`), then **restart your terminal**.

## Setup

```bash
cd david-portfolio
npm install
npm run dev
```

Open http://localhost:3000.
